# Audio Files

Letakkan file audio di folder ini:

## Required Files:
- **point.mp3** - Suara saat melewati obstacle (dapat poin)
- **death.mp3** - Suara saat game over
- **jump.mp3** - Suara saat lompat
- **buff.mp3** - Suara saat mendapat power-up
- **bgm.mp3** - Background music (akan diputar berulang)

## Format:
- Format MP3 direkomendasikan
- Ukuran file kecil (< 500KB per file, kecuali bgm)
- BGM bisa lebih besar (< 5MB)

## Tips:
- Gunakan suara pendek untuk point, jump, buff, death
- BGM sebaiknya seamless loop
